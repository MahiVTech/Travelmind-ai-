# 🌟 Vision

Travel today means constantly switching between apps.

📍 Google Maps

🍽 Food Apps

🚕

Cab Apps

🌦 Weather

📄 Documents

💱 Currency

🤖 AI

TravelMind AI brings these experiences together into **one modern interface**, making travel planning simpler, faster, and more enjoyable.

---

# 🚀 Features

| Feature | Description |
|----------|-------------|
| 🤖 AI Assistant | AI-inspired travel guidance and trip planning |
| 🛡 Safety Hub | Safety score, emergency numbers, travel tips |
| 🗺 Hidden Places | Discover lesser-known attractions |
| 🍽 Food Explorer | Local restaurant recommendations |
| 🌦 Weather | Weather overview *(ready for live API integration)* |
| 💰 Budget Planner | Organize trip expenses |
| 🚕 Smart Transport | Cab and transport suggestions |
| 📄 Documents | Keep travel information organized |
| 🚨 SOS Mode | Emergency contacts and quick actions |
| 📷 AI Scanner | Foundation for menu/document scanning |
| 💱 Currency | Currency conversion utilities |
| ⚙ Settings | Personalization and preferences |

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/MahiVTech/Travelmind-ai-.git
```

Move inside the project

```bash
cd Travelmind-ai-
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```
